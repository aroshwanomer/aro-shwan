Hello everyone, today I will present my project about Music Dataset Analysis using Python and Google Colab.

First, I used Google Colab to run Python code online.
Google Colab allows us to write and execute Python without installing anything on the computer.

Then I imported important libraries:

Pandas for datasets
NumPy for calculations
Matplotlib for charts and visualization.

After that, I loaded my music dataset CSV file.
The dataset contains more than 550 rows and multiple columns.
Three columns are numerical:

Streams
Duration
Popularity score.

Next, I used df.info() to display dataset information such as rows, columns, and data types.

Then I used describe() for descriptive statistics.
This function shows:

mean
minimum
maximum
standard deviation.

Mean means the average value.
Median means the middle value after sorting numbers.
Mode means the most repeated value.

I also calculated:

variance
standard deviation
quartiles.

Quartiles divide the data into four equal parts:

Q1 is 25%
Q2 is the median
Q3 is 75%.

After that, I created charts like:

bar chart
histogram
pie chart
box plot.

These charts help visualize the music dataset and understand the data better.

Finally, I uploaded my project to GitHub to store and share it online.

Thank you.”
